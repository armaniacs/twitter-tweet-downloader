# Twitter Tweet Downloader (Chrome Extension)

Twitter (X) のツイートをダウンロードし、TSV/Markdown形式で保存するChrome拡張機能です。

> **Note**: 古いPython/Goスクリプト版のドキュメントは [LEGACY_README.md](LEGACY_README.md) を参照してください。

## 特徴

- 📅 **期間指定**: 今日、昨日、今週などの期間を指定してダウンロード
- 🔄 **続きから取得**: 前回ダウンロードした日時の続きから再開 (Resume機能)
- 🎯 **個別ツイート**: 表示中の単一ツイートを即座にMarkdown化
- 📋 **出力**: コピペしやすいタブ区切りテキスト (TSV)。リンクはMarkdown形式 `[text](url)` で保持

## インストール方法

本拡張機能は開発者モードを使ってインストールします。

1. このリポジトリを適当な場所にクローンまたはダウンロードします。
2. Chromeを開き、 `chrome://extensions/` にアクセスします。
3. 右上の **デベロッパーモード** をONにします。
4. **パッケージ化されていない拡張機能を読み込む** をクリックします。
5. リポジトリ内の `chrome_extension` フォルダを選択します。

## 使い方

拡張機能のアイコンをクリックしてポップアップを開きます。

### プロファイルモード（一括ダウンロード）

Twitterのユーザープロフィールページ（例: `https://x.com/username`）を開いた状態で使用します。

1. **期間を選択**: "Today", "Yesterday" などを選択します。
   - "Custom Range" で日付指定も可能です。
   - **"Resume from last download"**: 前回実行した続きから取得したい場合にチェックを入れます。
2. **"Extract Tweets"** をクリックします。
3. 自動的にスクロールが行われ、ツイートが収集されます。
   - ⚠️ **注意**: 抽出中はポップアップを閉じないでください。閉じると処理が中断されます。
4. 完了するとテキストボックスに結果が表示されます。
5. **"Copy to Clipboard"** でコピーします。

### 個別ツイートモード

個別のツイートページ（例: `https://x.com/username/status/123...`）を開いた状態で使用します。

1. 拡張機能アイコンをクリックします。
2. **"Extract Tweets"** をクリックします（期間設定は無視されます）。
3. 表示中のツイートがMarkdown化され表示されます。

## 出力フォーマット

出力はTSV（タブ区切り）形式です。

```
実行時刻 <TAB> 【X】 <TAB> ツイート日時 <TAB> ツイート本文
```

例:
```
10:00	【X】	12/25 09:30	これはテストツイートです。[リンク](https://x.com)
```

## トラブルシューティング

- **ポップアップが閉じてしまう**: 抽出処理中はポップアップを開いたままにする必要があります。
- **ツイートが見つからない**: ページ読み込みが完了しているか確認してください。
- **ログイン**: Twitterにログインしている必要があります。
